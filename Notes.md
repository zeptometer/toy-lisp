# 今回の収録について
Toy Lisp

## 目標
* Lisp処理系を作る
* fibonacci を動かせるようにする

```
; f(0) = 0, f(1) = 1, f(2) = 1, ...
(define fib (lambda (x)
    (if (<= x 1) x
        (+ (fib (- x 1)) (fib (- x 2)))))
```


## 要求
* `define` (変数定義)
* `lambda` (無名関数)
* 変数
* `if`
* 数値計算 (整数) ← fixed int
* 再帰
* REPL (Read-Evaluate-Print Loop)
> # REPLとは、プログラミング言語の実行環境の一つで、利用者が入力欄にキーボードなどから式や文を一行入力すると、即座に解釈・実行して結果を返し、再び入力可能になるもの。
